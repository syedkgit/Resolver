## Framework Description:

This is a Selenium Page Object Model framework
This framework is created with Selenium WebDriver (Java), Maven, TestNG, WebDriverManager, Extent Reports.

### Good to know

- WebPage is QEIndex HTML file which can be placed in any directory and the directory address can be used as URL placed in _config.properties_
- User can input the browser under scope for testing in _config.properties_
- Objects for each test scenario is created in separate class for better readability
- Runner Log is used for logging the progress of test execution
- JS scroll is used to focus on the Test elements during execution
